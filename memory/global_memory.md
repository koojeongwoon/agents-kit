# Developer Global Preference & Persona Memory

이 파일은 개발자의 개인 코딩 선호도, 아키텍처 성향, 습관을 모든 AI 클라이언트에 보존하는 단일 전역 개발자 메모리입니다.

## 1. 개발 환경 & 기술 스택 선호
- **Frontend**: HTML5, Vanilla JavaScript, Vanilla CSS, Vite (TailwindCSS는 명시 요청 시만)
- **Backend & Tooling**: Node.js, Express, ESNext, Rust (Tauri v2)
- **Code Style**: 과도한 래핑 없이 명확하고 간결하며 신뢰성 높은 커스텀 패턴 작성
- **Testing**: 프로덕션 빌드 번들 및 런타임 오류 검증 최우선

## 2. 작업 & 대화 방식 선호
- 기존 코드 스타일 및 네이밍 컨벤션을 최우선 준수
- 주석 및 설명문은 필요한 핵심 요구사항만 명확하게 요약
